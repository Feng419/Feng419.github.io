迭代策略评估（Iterative Policy Evaluation）

我们对于策略的评估是通过当前策略 π\piπ 的价值函数 VπV_{\pi}Vπ​ 来决定的。对于 VπV_{\pi}Vπ​ ，我们用反向迭代的方式来求：

  • 在第 k+1k + 1k+1 次迭代中
  • 对于 ∀s∈S\forall s \in \mathcal{S}∀s∈S
  • 从 vk(s′)v_k(s')vk​(s′) 更新 vk+1(s)v_{k + 1}(s)vk+1​(s) ，其中 s′s's′ 是 sss 的上一状态

此方法最后会收敛至 VπV_{\pi}Vπ​ ，收敛性在后面会证明。

由贝尔曼方程可知：

vk+1(s)=∑a∈Aπ(a∣s)(Rsa+γ∑s′∈SPss′avk(s′))v_{k+1}(s) = \sum_{a\in \mathcal{A}}\pi (a|s) \left ( \mathcal{R}^a_s + \gamma \sum_{s' \in \mathcal{S}} \mathcal{P}^a_{ss'} v_{k}(s') \right ) vk+1​(s)=a∈A∑​π(a∣s)(Rsa​+γs′∈S∑​Pss′a​vk​(s′))

其矩阵形式如下：

vk+1=Rπ+γPπvkv^{k + 1} = \mathcal{R}^{\pi} + \gamma \mathcal{P}^{\pi} v^k vk+1=Rπ+γPπvk

策略迭代（Policy Iterative）

通过策略评估的方式，我们可以在每一步行动之后贪心的去更新策略。即：

  • 评估策略

vπ=E[Rt+1+γRt+2+⋯∣St=s]v_{\pi} = \mathbb{E}[R_{t + 1} + \gamma R_{t + 2} + \cdots | S_t = s] vπ​=E[Rt+1​+γRt+2​+⋯∣St​=s]

  • 贪心的改进策略

π′=greedy(vπ)\pi' = \text{greedy}(v_{\pi}) π′=greedy(vπ​)

可以证明的是，这个过程最后会收敛于 π∗\pi_*π∗​ 。

策略改进（Policy Improvement）

我们考虑一个确定的策略，a=π(s)a = \pi(s)a=π(s) 。
依据上文，我们可以贪心地改进策略，即：

π′(s)=arg max⁡a∈Aqπ(s,a)\pi'(s) = \argmax _{a\in \mathcal{A}} q_{\pi} (s,a) π′(s)=a∈Aargmax​qπ​(s,a)

我们可以证明它确实改进了价值函数：

qπ(s,π′(s))=max⁡a∈Aqπ(s,a)≥qπ(s,π(s))=vπ(s)vπ≤qπ(s,π′(s))=Eπ′[Rt+1+γvπ(St+1)∣St=s]≤Eπ′[Rt+1+γqπ(St+1,π′(St+1))∣St=s]≤Eπ′[Rt+1+γRt+2+γ2qπ(St+2,π′(St+2))∣St=s]≤Eπ′[Rt+1+γRt+2+⋯∣St=s]=vπ′(s)q_\pi(s, \pi'(s)) = \max_{a\in \mathcal{A}}q_\pi(s, a) \geq q_\pi(s, \pi(s)) = v_\pi(s) \\ \begin{align*} v_\pi &\leq q_\pi(s, \pi'(s)) = \mathbb{E}_{\pi'}[R_{t + 1} + \gamma v_\pi(S_{t+ 1})|S_t = s] \\ &\leq \mathbb{E}_{\pi'}[R_{t + 1} + \gamma q_\pi(S_{t + 1}, \pi'(S_{t + 1}))|S_t = s] \\ &\leq \mathbb{E}_{\pi'}[R_{t + 1} + \gamma R_{t + 2} + \gamma^2q_\pi(S_{t + 2},\pi'(S_{t + 2}))|S_t = s] \\ &\leq \mathbb{E}_{\pi'}[R_{t + 1} + \gamma R_{t + 2} + \cdots |S_t = s] = v_{\pi'}(s) \\ \end{align*} qπ​(s,π′(s))=a∈Amax​qπ​(s,a)≥qπ​(s,π(s))=vπ​(s)vπ​​≤qπ​(s,π′(s))=Eπ′​[Rt+1​+γvπ​(St+1​)∣St​=s]≤Eπ′​[Rt+1​+γqπ​(St+1​,π′(St+1​))∣St​=s]≤Eπ′​[Rt+1​+γRt+2​+γ2qπ​(St+2​,π′(St+2​))∣St​=s]≤Eπ′​[Rt+1​+γRt+2​+⋯∣St​=s]=vπ′​(s)​

当Improvement停止后，

qπ(s,π′(s))=max⁡a∈Aqπ(s,a)=qπ(s,π(s))=vπ(s)q_\pi(s, \pi'(s)) = \max _{a\in \mathcal{A}}q_\pi(s, a) = q_\pi (s, \pi(s)) = v_\pi(s) qπ​(s,π′(s))=a∈Amax​qπ​(s,a)=qπ​(s,π(s))=vπ​(s)

vπ(s)=max⁡a∈Aqπ(s,a)(1)v_\pi(s) = \max_{a\in \mathcal{A}} q_\pi(s, a) \tag1 vπ​(s)=a∈Amax​qπ​(s,a)(1)

由于最优化原则告诉我们：

如果策略 π\piπ 是最优策略，当且仅当对于 π\piπ 中状态 sss 所能达到的任意状态 s′s's′ 均有 vπ(s′)=v∗(s)v_\pi(s') = v_*(s)vπ​(s′)=v∗​(s)
